## [1.1.0] - 2026-03-27

### 新增 / Added
- ✨ **多语言支持** / **Multi-language Support**
  - 支持简体中文（zh_CN）和英文（en_US）
  - Support for Simplified Chinese (zh_CN) and English (en_US)

- 🌐 **国际化模块** / **Internationalization Module**
  - 新增 `i18n.py` 模块，提供完整的国际化功能
  - Added `i18n.py` module with complete i18n functionality

- 🎨 **语言选择器** / **Language Selector**
  - GUI界面添加语言下拉选择框
  - Added language dropdown selector in GUI

- 📝 **双语文档** / **Bilingual Documentation**
  - 新增英文版 README (`README_EN.md`)
  - Added English README (`README_EN.md`)
  - 新增双语使用说明 (`USAGE.md`)
  - Added bilingual usage guide (`USAGE.md`)

- 🔧 **配置持久化** / **Configuration Persistence**
  - 语言设置自动保存到 `config.json`
  - Language settings auto-saved to `config.json`

### 改进 / Improved
- 📊 **Excel报告国际化** / **Excel Report Internationalization**
  - 所有工作表名称支持多语言
  - All worksheet names support multiple languages
  - 表头和标签完全国际化
  - Headers and labels fully internationalized

- 🎨 **字体自适应** / **Adaptive Fonts**
  - 中文使用"微软雅黑"，英文使用"Arial"
  - Chinese uses "Microsoft YaHei", English uses "Arial"

- 💬 **消息框国际化** / **Message Box Internationalization**
  - 所有提示信息支持多语言
  - All prompts support multiple languages

### 技术细节 / Technical Details
- 实现单例模式的国际化管理器
- Implemented singleton pattern for i18n manager
- 支持运行时语言切换（需重启生效）
- Support runtime language switching (requires restart)
- 完整的翻译键值对系统
- Complete translation key-value system

## [1.0.0] - 2026-03-26

### 新增 / Added
- 🎉 **初始版本发布** / **Initial Release**

- 📊 **ELF文件解析** / **ELF File Parsing**
  - 支持GHS编译器生成的ELF文件
  - Support for ELF files generated by GHS compiler
  - 自动分类内存段（ROM、RAM、Stack、Heap）
  - Automatic memory section classification (ROM, RAM, Stack, Heap)

- 📈 **Excel报告生成** / **Excel Report Generation**
  - 内存使用摘要表
  - Memory usage summary sheet
  - 详细段信息表
  - Detailed sections sheet
  - ROM/RAM详细信息表
  - ROM/RAM details sheets

- 🔍 **符号表解析** / **Symbol Table Parsing**
  - 统计每个段的实际用量
  - Calculate actual usage for each section
  - 计算剩余容量
  - Calculate remaining capacity
  - 区间合并算法处理符号重叠
  - Interval merging algorithm for symbol overlap

- 📍 **地址空隙检测** / **Address Gap Detection**
  - 自动检测内存映射中的地址空隙
  - Automatically detect address gaps in memory map
  - 在报告中显示空隙信息
  - Display gap information in reports

- 🖥️ **图形用户界面** / **Graphical User Interface**
  - 友好的tkinter界面
  - User-friendly tkinter interface
  - 芯片型号快速选择
  - Quick chip model selection
  - 自定义ROM/RAM容量配置
  - Custom ROM/RAM capacity configuration

- 📦 **芯片配置预设** / **Chip Configuration Presets**
  - R7F701372A (ROM: 4MB, RAM: 480KB)
  - R7F701374A (ROM: 2MB, RAM: 320KB)
  - R7F701382 (ROM: 1MB, RAM: 192KB)
  - R7F701312 (ROM: 1MB, RAM: 128KB)

### 技术特性 / Technical Features
- 基于 pyelftools 的ELF解析
- ELF parsing based on pyelftools
- 基于 openpyxl 的Excel生成
- Excel generation based on openpyxl
- 多线程后台分析
- Multi-threaded background analysis
- 完整的错误处理
- Complete error handling
