## 中文

### 1. 安装

```bash
# 克隆或下载项目
git clone <repository-url>
cd MemoryMapTool

# 安装依赖
pip install -r requirements.txt
```

### 2. 启动程序

```bash
python main.py
```

### 3. 使用步骤

#### 步骤 1: 选择语言
- 在界面右上角的语言下拉框中选择语言
- 选择 `zh_CN`（简体中文）或 `en_US`（English）
- 重启程序使语言生效

#### 步骤 2: 选择ELF文件
- 点击"ELF文件"旁的"浏览..."按钮
- 选择GHS编译器生成的ELF文件（.elf 或 .out）

#### 步骤 3: 设置输出路径（可选）
- 程序会自动生成输出路径
- 也可以点击"输出Excel"旁的"浏览..."按钮自定义路径

#### 步骤 4: 配置芯片信息（可选）
- **方式1**: 从下拉框选择预设的芯片型号
  - R7F701372A (ROM: 4MB, RAM: 480KB)
  - R7F701374A (ROM: 2MB, RAM: 320KB)
  - R7F701382 (ROM: 1MB, RAM: 192KB)
  - R7F701312 (ROM: 1MB, RAM: 128KB)

- **方式2**: 手动输入ROM/RAM总容量（单位：KB）

#### 步骤 5: 开始分析
- 点击"开始分析"按钮
- 等待分析完成
- 查看生成的Excel报告

### 4. 查看报告

生成的Excel文件包含4个工作表：

1. **内存使用摘要**
   - ROM、RAM、Stack、Heap的总体使用情况
   - 包含空隙统计

2. **详细段信息**
   - 所有内存段的详细信息
   - 显示地址空隙

3. **ROM详细信息**
   - ROM相关段的详细分解

4. **RAM详细信息**
   - RAM相关段的详细分解

### 5. 常见问题

**Q: 分析失败怎么办？**
- 检查ELF文件是否完整
- 确保输出文件未被其他程序打开
- 查看错误提示信息

**Q: 如何获取ROM/RAM总容量？**
- 查看芯片数据手册
- 查看链接脚本中的MEMORY定义
- 可以留空，工具仍会显示实际使用量

**Q: 语言切换后没有生效？**
- 需要重启程序才能完全生效

## English

### 1. Installation

```bash
# Clone or download the project
git clone <repository-url>
cd MemoryMapTool

# Install dependencies
pip install -r requirements.txt
```

### 2. Launch Program

```bash
python main.py
```

### 3. Usage Steps

#### Step 1: Select Language
- Choose language from the dropdown in the top-right corner
- Select `zh_CN` (Simplified Chinese) or `en_US` (English)
- Restart the program for the language change to take effect

#### Step 2: Select ELF File
- Click "Browse..." button next to "ELF File"
- Select an ELF file generated by GHS compiler (.elf or .out)

#### Step 3: Set Output Path (Optional)
- The program will auto-generate an output path
- You can also click "Browse..." next to "Output Excel" to customize the path

#### Step 4: Configure Chip Information (Optional)
- **Method 1**: Select a preset chip model from the dropdown
  - R7F701372A (ROM: 4MB, RAM: 480KB)
  - R7F701374A (ROM: 2MB, RAM: 320KB)
  - R7F701382 (ROM: 1MB, RAM: 192KB)
  - R7F701312 (ROM: 1MB, RAM: 128KB)

- **Method 2**: Manually enter ROM/RAM total capacity (in KB)

#### Step 5: Start Analysis
- Click "Start Analysis" button
- Wait for analysis to complete
- View the generated Excel report

### 4. View Report

The generated Excel file contains 4 worksheets:

1. **Memory Summary**
   - Overall usage of ROM, RAM, Stack, Heap
   - Includes gap statistics

2. **Detailed Sections**
   - Detailed information for all memory sections
   - Shows address gaps

3. **ROM Details**
   - Detailed breakdown of ROM-related sections

4. **RAM Details**
   - Detailed breakdown of RAM-related sections

### 5. FAQ

**Q: What if analysis fails?**
- Check if the ELF file is complete
- Ensure the output file is not opened by another program
- Check the error message

**Q: How to get ROM/RAM total capacity?**
- Check chip datasheet
- Check MEMORY definition in linker script
- Can leave blank, tool will still show actual usage

**Q: Language change not taking effect?**
- Need to restart the program for changes to take full effect
